What I learned

  1. I have learned what CSS means. It means Cascading Style Sheets. CSS adds style to the webste, making it easier or harder to view depending on the purpose of the website.

Lab 5

Published on October 15th, 2024 by Kaden Duong

What I learned

  1. I have learned what the different CSS layouts are. It includes Normal Flow, Flexbox, and Grid. The normal flow has no CSS to it and just follows the order of the code from top to bottom. The Flexbox layout makes it easier to implement more resonsive dynamic layouts. The Grid layout allows you to create a 2D layout consisting of rows and columns.
  2. Another thing I learned is that Media queries are a feature in CSS that allow you to apply different styles based on certain conditions, typically related to the characteristics of the device (phone, tablet, desktop) or browser viewing the web page.
  3. Furthermore, I learned more about configuring margin, padding, and borders in an in-depth way.

Learning about how layouts are made, media queries, and diving more into margin, padding, and borders was realy helpful for building on from the basics of CSS. I learned a lot from:

  • CSS Grid Layout Guide
  • CSS Flexbox Layout Guide
