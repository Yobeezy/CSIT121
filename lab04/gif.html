File Type

This image is a GIF image. GIF stands for Graphics Interchange Format. The animated GIF has made and kept this format popular, but its format can be used for still graphics as well. Some GIF facts:

  • Best used for line art and logos
  • Maximum of 256 colors
  • One color can be configured as transparent
  • Can be animated
  • Uses lossless compression (data compression that reduces file sizes without sacrificing any significant information in the process)
  • Can be interlaced (Displays low-resolution versions of the image in a browser as the file downloads. Interlacing makes download time seem shorter but also increases the file size.)
